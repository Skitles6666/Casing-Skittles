# Casing-Skittles

A free casino game collection built in Godot.

## Games

- Keno
- Craps
- Roulette
- Baccarat
- Blackjack
- Video poker
- Pai Gow Poker
- Three Card Poker
- Slots

## Run It

1. Install Godot 4.
2. Open this folder as a Godot project.
3. Run `KenoGame.tscn`.

This is a play-money casino project for entertainment and learning. It does not support real-money gambling.
